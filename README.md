# Finance Projects

A comprehensive collection of finance-related applications and tools built with modern web technologies.

## Projects Overview

### 📊 Budget Tracker
- **Description**: Personal budget management application with income/expense tracking 
- **Features**: Monthly budgeting, category-wise spending, visual analytics
- **Tech Stack**: React, TailwindCSS, Chart.js

### 📈 Stock Portfolio Analyzer
- **Description**: Real-time stock portfolio tracking and analysis tool
- **Features**: Portfolio performance, gain/loss calculations, stock price tracking
- **Tech Stack**: React, Financial APIs, Recharts

### 💰 Expense Categorizer
- **Description**: Automated expense categorization using ML techniques
- **Features**: Smart categorization, receipt scanning, spending patterns
- **Tech Stack**: Python, Flask, TensorFlow

### 🧮 Financial Calculator Suite
- **Description**: Collection of financial calculation tools
- **Features**: Loan calculator, investment returns, retirement planning
- **Tech Stack**: Vanilla JavaScript, HTML5, CSS3

## Getting Started

Each project is self-contained with its own README and setup instructions. Navigate to individual project directories to get started.

## Technologies Used

- **Frontend**: React, HTML5, CSS3, TailwindCSS
- **Backend**: Python, Flask, Node.js
- **Data Visualization**: Chart.js, Recharts, D3.js
- **APIs**: Yahoo Finance, Alpha Vantage, Plaid
