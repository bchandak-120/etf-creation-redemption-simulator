# GitHub Pages Setup Instructions

## ✅ Files Successfully Pushed!

## 🌐 Enable GitHub Pages:

1. **Go to your repository**: https://github.com/bchandak-120/portfolio-webstite
2. **Click "Settings" tab**
3. **Scroll down to "Pages" section** (left sidebar)
4. **Under "Build and deployment":**
   - Source: Select "Deploy from a branch"
   - Branch: Select "main"
   - Folder: Select "/ (root)"
5. **Click "Save"**

## 🚀 Your Live Portfolio Links:

After enabling Pages, your portfolio will be live at:

### Main Portfolio (Full Version):
```
https://bchandak-120.github.io/portfolio-webstite/my-portfolio/
```

### Simplified Portfolio:
```
https://bchandak-120.github.io/portfolio-webstite/my-portfolio/portfolio.html
```

## ⏱️ Note:
GitHub Pages may take 1-2 minutes to activate after saving.

## 🔧 If Cascade preview doesn't work:
- The GitHub Pages link will work perfectly
- Direct file access: Double-click `portfolio.html` in your folder
